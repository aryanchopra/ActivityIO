## Run Locally

### Clone the project

```bash
  git clone https://github.com/aryanchopra/ActivityIO
```

### Go to the project directory

```bash
  cd ActivityIO
```


### Install dependencies


```bash
  npm install && cd client && npm install
```


### Environment Variables

To run this project, you will need to add the following environment variables to your .env file located in the root directory:  
`PORT`  
`MONGODB_URL` : MongoDB Connection String  
`SECRET` : Randing String for JWT Authentication   
and `REACT_APP_GOOGLECLIENTID` in a .env file located in the root of the client folder

### API Keys
You can get your Google Client API key by logging in to your Google account at https://console.cloud.google.com/apis/dashboard and creating a new application.


### Start the server

```bash
  npm run dev
```

### Start the client

```bash
  cd client && yarn start
```
